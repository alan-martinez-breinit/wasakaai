# WasakaAI

> Otro tú. Sin límites.

Segundo cerebro personal potenciado por IA. No es una herramienta — es **otro Alan** con potencia de IA.

## Qué es

- Sistema de conocimiento personal que captura, conecta y genera
- Privado, en la nube vía OpenCode
- Basado en Obsidian + OpenCode Go (JARVIS)
- Estructura PARA+ expandida

## Principios

1. Todo se registra — pero siempre se pregunta antes
2. Privado y en la nube — vía OpenCode
3. Sin límites — no hay zonas excluidas
4. Nada se borra — se archiva
5. Bilingüe — español para ti, inglés para lo técnico
6. Proactivo pero respetuoso — propone, no ejecuta ciegamente
7. Otro tú — aprende, conecta, genera desde tu conocimiento

## Stack

| Componente | Herramienta |
|-----------|-------------|
| Interfaz | Obsidian |
| Agente | JARVIS (OpenCode Go) |
| LLM | OpenCode Cloud |
| Embeddings | servicio cloud |
| Búsqueda semántica | ChromaDB |
| Captura web | Obsidian Web Clipper |
| Transcripción | Whisper (local) |
