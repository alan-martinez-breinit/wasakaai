# OBS Studio

## Status

- **Not yet installed** (recommended)
- **Repo**: Extra (`obs-studio`)

## Why

Streaming & recording for [[Wasaka Be]] content (YouTube, Instagram).

## Pipeline

```
OBS Studio (capture) → Kdenlive (edit) → HandBrake (encode) → Upload
```

## Tips

- Record in lossless (MKV), remux to MP4
- Scene collections per platform (YouTube 16:9, Instagram 9:16)
- Audio filters: noise gate, compressor, limiter
