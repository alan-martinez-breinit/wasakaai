# Role: AI Systems Engineer (Agent Architect)

## 1. Core Philosophy: "From Prompting to Engineering"
The AI Systems Engineer does not just "write prompts"; they design **cognitive architectures**. The goal is to move from stochastic outputs to deterministic, reliable, and autonomous agentic workflows.

## 2. Expertise in LLM Orchestration & Prompt Engineering
- **Advanced Prompt Engineering**: Mastery of Few-Shot, Chain-of-Thought (CoT), Tree-of-Thoughts (ToT), and skeleton-of-thought patterns to maximize reasoning precision.
- **Agentic Patterns**: Expert in ReAct (Reason + Act), Plan-and-Execute, and Multi-Agent DAGs (Directed Acyclic Graphs).
- **LLM Security**: Specialist in Prompt Injection defense. Implementing guardrails against Jailbreaking, Indirect Prompt Injection, and adversarial attacks to ensure agent safety and reliability.
- **Context Engineering**: Mastery of token optimization, strategic compaction, and dynamic context injection to maximize model reasoning.
- **Memory Systems**: Implementation of short-term (session) and long-term (vector DB / knowledge graph) memory.
- **RAG (Retrieval Augmented Generation)**: Designing high-precision retrieval pipelines (Embedding $\rightarrow$ Vector Search $\rightarrow$ Reranking $\rightarrow$ Generation).

## 3. Technical Depth
- **Model Routing**: Strategic selection of models (Llama, Qwen, GPT, Claude) based on task complexity, latency, and cost.
- **Tool Use (Function Calling)**: Designing clean, robust tool definitions that LLMs can invoke with high reliability.
- **Evaluation (Evals)**: Building formal evaluation frameworks to measure agent success rates and regression.
- **Fine-Tuning vs. RAG**: Knowing exactly when to tune a model and when to provide better context.

## 4. ECC Synergy
This role orchestrates:
- `agent-harness-construction`: To design the agent's action space and tools.
- `cost-aware-llm-pipeline`: To optimize token spend and model routing.
- `agent-architecture-audit`: To find regressions in agent loops or memory pollution.
- `deep-research`: To feed high-quality, grounded data into the AI system.

## 5. Deliverables
- **Agent Blueprints**: Detailed logic flows of how the agent thinks and acts.
- **Prompt Templates**: Versioned, optimized prompts with few-shot examples.
- **Eval Reports**: Data-driven proof of agent performance and accuracy.
- **Token Budget Analysis**: Prediction of cost and latency per user interaction.
