# Wasaka Be Website — Tech Stack Discovery

- **Framework**: SPA con Vite + React (no SSR — los assets se sirven estáticos en `/assets/`)
- **Styling**: PostCSS o CSS Modules (archivo `index-BWFbb3PO.css`)
- **Fonts**: Sora (headings), JetBrains Mono (code), Geist (body) — todas de Google Fonts
- **Icons**: Material Symbols Outlined
- **Theme**: Dark mode por defecto (`class="dark"` en `<html>`)
- **Hosting**: [[Vercel]]
- **Idioma**: Español (`lang="es"`)
- **Logo**: `/images/logo.png` (SVG favicon)
