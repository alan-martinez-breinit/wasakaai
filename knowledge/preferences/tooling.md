# Operational & Tooling Preferences

## Technology Stack
- **OS Philosophy**: Bleeding-edge, maximum customization (Arch Linux + KDE)
- **Deployment**: Vercel for web projects
- **Containers**: Docker for local dev
- **Editors**: Vim for quick edits, OpenCode for AI-assisted work
- **Package Management**: Prefers pacman/AUR over Flatpak/Snap

## Workflow & Knowledge
- **Knowledge Management**: Building a local knowledge graph (Obsidian-compatible)
- **Content Creation**: YouTube + Instagram + Facebook (multi-platform)
- **Branding**: Manages two brands ([[Wasaka Be]], [[New Nuwamei]])

## Corporate Policy (BREINIT)
- **Read-Only Mode**: WasakaAI no modifica nada sin autorización explícita del usuario
- **No crear/editar/eliminar archivos** sin permiso
- **No ejecutar comandos con efectos secundarios** sin aprobación (install, delete, config change, registry, network)
- **No instalación/desinstalación de software**
- **Diagnóstico, análisis y consulta**: siempre permitidos sin restricción
- **Excepción**: Esta base de conocimiento (knowledge graph) puede actualizarse automáticamente, pues es memoria interna del asistente
