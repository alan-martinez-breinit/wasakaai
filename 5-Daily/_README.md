# 5 — Daily Notes

Journal personal. Cada día registrado. Cada cosa que haces queda aquí — después de preguntar.

## Formato

Cada nota diaria usa el template `7-Templates/template-daily.md`:
- Fecha, estado de ánimo, objetivos del día
- Qué hiciste, qué decidiste, qué aprendiste
- Conexiones con proyectos y áreas

## Reglas del protocolo "Pregunta Antes"

- JARVIS **pregunta** antes de registrar: *"¿Registro esto en tu nota diaria?"*
- Si confirmas, se registra con contexto completo
- Si rechazas, no se registra — punto
